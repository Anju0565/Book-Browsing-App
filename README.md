# Book-Browsing-App
To run this project, 
<ol>
  <li>cd client/cd server</li>
  <li>npm install</li>
  <li>npm start</li>
</ol>

<h3>Features:<h3/>
  <ol>
    <li>Homepage</li>
    <li>Sign up</li>
     <li>Login Page</li>
      <li>Book Listing</li>
      <li>Add New Book</li>
      <li>Search Book</li>
      <li>Filter By Name, Author, Price</li>
     <li>Delete Listed Book</li>
  </ol>
